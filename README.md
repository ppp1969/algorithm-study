# Algorithm Study

코딩 테스트 대비를 위한 알고리즘 스터디 레포지토리입니다.  
문제 풀이와 코드 리뷰를 통해 다양한 알고리즘 패턴을 학습하고,  
핵심 유형을 익힌 뒤 실전 기출 문제로 확장하는 것을 목표로 합니다.

---

# 🎯 Study Goal

* 알고리즘 문제 해결 능력 향상
* 자주 등장하는 핵심 알고리즘 유형 학습
* 코드 리뷰를 통한 사고 과정 공유
* 지속적인 문제 풀이 습관 형성
* 실전 기출 문제 대응력 강화

---

# 🗺️ Problem Solving Roadmap

이 스터디는 단순히 문제를 많이 푸는 것보다,  
**핵심 유형을 익히고 실전 기출로 확장하는 과정**을 목표로 합니다.

### Step 1. Programmers 고득점 Kit

고득점 Kit를 통해 자주 나오는 알고리즘 유형을 먼저 학습합니다.

* Hash
* Stack / Queue
* Sort
* Exhaustive Search
* Greedy
* Heap
* DFS / BFS
* Binary Search
* Dynamic Programming
* Graph

### Step 2. 실전 기출 문제

고득점 Kit 완주 후에는 실제 기출 세트를 중심으로 풀이를 이어갑니다.

* 2024 KAKAO WINTER INTERNSHIP
* 2023 KAKAO BLIND RECRUITMENT
* 2022 KAKAO TECH INTERNSHIP
* 2022 KAKAO BLIND RECRUITMENT
* 2023 Dev-Matching
* 2022 Dev-Matching
* 2021 Dev-Matching
* PCCP 기출문제

# 🤝 Collaboration Guide


* 더 효율적인 알고리즘이 있는지
